    1.Hexo安装

    需要事先安装好：

    • nodejs，可以使用node -v查看是否已经安装

    • git，可以使用git --version查看是否已经安装

      一般下载完之后还需要将其配置到系统路径中，参考这一篇

    确保都安装好后：

    • 新建一个空文件夹，cd进去

    • 可以右键打开powershell，也可以使用Git Bash，按顺序执行以下命令

      • npm install -g hexo

      • hexo init 你的文件夹名

        初始化成功后会显示Start blogging with Hexo!，同时空文件夹下会多出一个你刚刚命名的文件夹

      输入的是hexo init akaneovoBlog

    • cd进入到新创建的文件夹中，按顺序执行以下命令

      • npm install
      • hexo g或hexo generate
      • hexo server

    • 此时会显示Hexo is running at http://localhost:4000/，可以打开浏览器查看，若展示出默认的博客，则安装成功

    3.Hexo-theme-Anatole

    因为默认的我真的不太喜欢，所以在Hexo官方的theme里面找了一个个人比较喜欢的hexo-theme-Anatole，它的github上就有安装和配置的相关文档，具体步骤为：

    • 下载主题

      • cd到theme目录下（就是刚刚init的文件夹里面）
      • git clone https://github.com/Ben02/hexo-theme-Anatole.git themes/anatole
      • cd anatole
      • git pull

      因为git clone之后还会有一个theme目录，就变成了双层，我就把里面那个anatole文件夹剪切出来放在原来的theme文件夹里面，然后把clone产生的文件夹删掉了
